同期型パズルゲーム

1つのキー入力で、2つの異なるマップにいるキャラクターを同時に動かし、両方のステージを同時にクリアするパズルゲームです。

公開URL
https://miyajimas.github.io/puzzle-game/

この作品のポイント（技術的工夫）
同期移動ロジックの実装：
  単一の入力に対して、2つの配列データ（マップ）を同時に参照し、壁や荷物の判定を独立して行いつつ、状態を更新するロジックを自力で構築しました。
データ駆動の描画：
  マップ情報を多次元配列で管理し、JavaScriptで動的にHTML（Table）を生成して描画する仕組みにしています。
ゲームバランスの設計：
  片方のキャラが壁にぶつかっている間にもう片方を動かすといった、ズレを利用して解くパズル性を盛り込みました。

使用技術
- HTML5
- CSS3 (Flexboxによるレイアウト)
- JavaScript (Vanilla JS)

開発の感想
2つの迷路の動きを同期させる処理で、片方がクリア条件を満たした時の判定や、複雑な壁の配置によるエラーに苦労しました。しかし、デバッグを繰り返すことで、配列操作と条件分岐の理解を深めることができ、最終的に意図した通りのパズル性を実現できました。
